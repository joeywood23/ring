Ring

Photorealistic 3D exploration of the San Francisco Bay Area

This app streams Google Photorealistic 3D Tiles, which requires a free Google Maps Platform API key:

  1. Go to Google Cloud Console → Maps Platform and create/select a project
  2. Enable the Map Tiles API
  3. Create an API key under Keys & Credentials

The key is stored only in your browser (localStorage) and sent only to Google.
